# Roster Manager Training Plan

*2 hours self-paced.*

## Module 1 — The delegated model (20 min)

**Read**
- [Rosters](../user/rosters.md)
- [Enrollments](../user/enrollments.md)

Understand:
- Which payers are delegated (Essen credentials on their behalf)
- The difference between roster-based submission and per-provider submission
- Monthly cycle and how NCQA expects rosters to be produced

## Module 2 — Drafting a roster (45 min)

**Practice**
1. Open the Rosters page. Find the current draft for Anthem.
2. Review every row — add, change, term. Understand why each is included.
3. For any row with a validation error, click to read the reason. Fix the underlying data on the provider record, then refresh validation.
4. Download the CSV. Open it. Review it against the payer's template.

## Module 3 — Submitting (30 min)

**Practice**
1. For an Availity payer (Anthem, Carelon): click `Submit`. Watch the status update.
2. For an email payer (Aetna): preview the email, review the attached CSV, click `Send`.
3. For a portal payer: download, manually upload to the payer portal (sandbox sim), and click `Mark submitted` with the confirmation number.

## Module 4 — Acknowledgements (15 min)

**Practice**
1. Upload a sample acknowledgement PDF to a prior submission.
2. For a rejected row, create a follow-up task. Observe the assignment to the provider's specialist.
3. Close an acknowledgement cleanly.

## Module 5 — Historical reporting (10 min)

**Practice**
1. Open Rosters → History and identify roster churn for the last 6 months.
2. Export a report of accepted vs. rejected rows by payer.

## Competency check

Pass rate: 85% on a 15-question assessment. Focus: cycle timing, validation resolution, per-payer submission paths, acknowledgement handling, roster history.
