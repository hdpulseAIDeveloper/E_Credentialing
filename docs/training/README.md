# Training

Role-based training plans for the ESSEN Credentialing Platform. Each plan lists the essential concepts, hands-on exercises, and competency check you must pass before you work independently.

## Plans by role

| Role | Duration | Plan |
|------|----------|------|
| Credentialing Specialist | 4 hours | [specialist.md](specialist.md) |
| Credentialing Manager | 3 hours | [manager.md](manager.md) |
| Committee Chair / Member | 90 minutes | [committee.md](committee.md) |
| Roster Manager | 2 hours | [roster-manager.md](roster-manager.md) |
| Compliance Officer | 2 hours | [compliance-officer.md](compliance-officer.md) |
| Administrator | 3 hours | [administrator.md](administrator.md) |

## Delivery

- Each plan is self-paced; materials link into the user guide and the live platform (sandbox tenant).
- Sandbox has synthetic providers you can edit freely without affecting production data.
- After completing, submit the competency check form linked at the bottom of each plan. An Admin records completion.

## Onboarding cadence

1. Day 1 — Microsoft 365 access + Azure AD group assignment
2. Day 2 — self-paced training plan
3. Day 3 — shadow a peer working real files
4. Day 4 — work 2–3 files with a peer pairing
5. Day 5 — competency check and sign-off

## Ongoing training

- **Quarterly** — 30-minute refresher covering new features and any policy updates.
- **Annual** — HIPAA and security training. A certificate is recorded in your user profile.
- **Ad-hoc** — when the platform adds a significant new feature, a short video is posted and announced.
