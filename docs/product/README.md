# Product Documentation — E-Credentialing CVO Platform

**Audience:** All stakeholders — leadership, customers, partners, sales,
new joiners. Avoids developer detail; explains *what* the platform does
and *why*.

The product is positioned as a **Credentialing Verification
Organization (CVO) platform**: an integrated stack that automates the
NCQA primary-source verification element catalog, OPPE / FPPE cycles,
payer enrollment, recredentialing, sanctions monitoring, and a public
FHIR R4 provider directory. See `/cvo` on the running app for the
customer-facing version of this story.
