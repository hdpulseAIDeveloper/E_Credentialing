## What this platform does

The platform supports the full credentialing lifecycle for healthcare providers practicing at Essen:

1. **Onboarding** — invite a provider, collect their application and supporting documents.
2. **Primary Source Verification (PSV)** — automatically verify licenses, DEA, boards, sanctions, and NPDB.
3. **Committee review** — prepare summary sheets, run committee sessions, record decisions.
4. **Enrollments** — submit providers to payers (Availity, My Practice Profile, Verity, eMedNY, and more).
5. **Expirables tracking** — monitor expiration dates and trigger provider outreach before anything lapses.
6. **Recredentialing** — run the 36-month cycle.
7. **Hospital privileges, OPPE/FPPE, telehealth, CME, and CV tracking** — full lifecycle support.
8. **Compliance and reporting** — NCQA CVO readiness and custom reports.
9. **Public REST/FHIR API** — integrate with downstream systems and payer directory requirements.
